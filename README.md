## CRYPTOCURRENT

# REAL TIME CRYPTO TRACKER

## To run this project

```javascript
   npm install  //To install dependencies
   npm start    // To launch it
```

ThIS PROJECT WAS BUILT FOR REAL TIME CRYPTO ANALYSIS WITH THE FOLLOWING:

- **REACT.JS**
- **SASS**
- **COINGECKO API**
- **CHARTJS**
- **CRYPTO-NEWS**
